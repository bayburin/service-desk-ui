Правила форматирования ответов

для переноса строки завершите предыдущую 2 пробелами

для создания пустой строки введите <br></br>

_курсив_ **жирный** или ~~зачеркнутый~~

одиночные кавычки экранируют `вот так`

для создания цитаты введите > в начале строки

пример создания ссылок

[example](https://example.com)

для создания блоков кода, используйте обратные апострофы ` или тильды ~

``` typescript
public markdown = "# Markdown";
const a = 123;
```
